- Desempaquete en un carpeta el contenido de este archivo ZIP.
- En el directorio base encontrará el archivo Futbol2006.bak, 
  que es una copia de seguridad de una base de datos de SQL
  Server 2005. Utilice SQL Server 2005 Management Studio para
  restaurarla.
- Los ejemplos se instalan cada uno en su directorio.
- Todos los ejemplos han sido probados con la versión definitiva
  de Visual Studio 2008.
- La aplicación Ejemplo11_05 (LinqToTFS) requiere que esté
  instalado el SDK de Visual Studio 2008. Adicionalmente, se
  deberá tener conexión a un servidor TFS y cambiar la URL
  (incrustada en el código del ejemplo) para poder ejecutarlo
  satisfactoriamente.
- Las aplicaciones de los capítulos 13 y 14 requieren que la 
  base de datos Futbol2006 esté instalada para funcionar.
- El resto de los ejemplos deberá compilar y ejecutar correc-
  tamente sin otro prerrequisito que el propio Visual Studio.
